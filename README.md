제미나이를 이용한 크롤링 내용 요약

동작 순서
1. 실행 시 제미나이 API Key 입력 하면 로컬에 API Key를 저장 하고 이 후 실행 시 API Key 를 불러옴

2. URL 입력 칸에 URL 입력 후 실행 시키면 파이썬 로컬 크롤링 + 제미나이가 크롤링을 시작함

3. 로컬 크롤링 결과와 제미나이 크롤링 결과를 다시 제미나이한테 주며 더 정확하고 완성도 높은 글을 선택하게 함

4. 선택 된 결과를 다시 제미나이 한테 주며 글의 요약을 시킴

5. 모든 동작이 완료 되면 로컬 크롤링, 제미나이 크롤링, 선택 된 크롤링 요약 본을 각각 다른 폴더에 저장
